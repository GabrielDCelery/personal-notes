# TypeScript Interview Prep

A concise, comprehensive refresher for experienced TypeScript/Node.js developers preparing for interviews.

## Lesson Series

### Package Management & Tooling
- [Lesson 01: package.json Deep Dive](lesson-01-package-json-deep-dive.md) - Dependencies, semver, exports, publishing workflow
- [Lesson 02: tsconfig.json Mastery](lesson-02-tsconfig-mastery.md) - Compiler options, module resolution, project references

### Type System Fundamentals
- [Lesson 03: Type System Internals](lesson-03-type-system-internals.md) - Inference, widening/narrowing, structural typing, variance

### Advanced Types
- [Lesson 04: Utility Types & Type Manipulation](lesson-04-utility-types-and-manipulation.md) - Built-in utilities, mapped types, conditional types, template literals
- [Lesson 05: Generics Deep Dive](lesson-05-generics-deep-dive.md) - Constraints, variance, default types, performance considerations

### Modules & Declarations
- [Lesson 06: Module Systems](lesson-06-module-systems.md) - ESM vs CommonJS, module resolution strategies, import/export patterns
- [Lesson 07: Declaration Files](lesson-07-declaration-files.md) - Writing .d.ts files, ambient declarations, module augmentation

### Publishing & Distribution
- [Lesson 08: Publishing npm Packages](lesson-08-publishing-npm-packages.md) - Build setup, dual publishing (ESM+CJS), versioning, distribution

## Target Audience

Developers with 2+ years TypeScript experience who need a focused refresher covering:
- What you might have forgotten
- Interview-critical concepts
- Common gotchas and edge cases
- Publishing and distribution best practices

## Usage

Each lesson includes:
- Quick reference tables for fast lookup
- Focused examples highlighting edge cases
- Interview questions you might encounter
- Practical challenges to test understanding
