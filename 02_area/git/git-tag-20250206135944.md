# Git tag

## Create tag with message

```sh
git tag -a v1.0.0 -m "your message"
```

## Push git tag to remote

```sh
git push origin tag v1.0.0
```

## Push a specific tag

```sh
git tag proto/v1.5.0
git push origin proto/v1.5.0
```

## Or push all tags at once

```sh
git push origin --tags
```

## Delete remote tag

```sh
git push origin --delete proto/v1.5.0
```

## Also delete locally if needed

```sh
git tag -d proto/v1.5.0
```
